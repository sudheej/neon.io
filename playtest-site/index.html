Quick Start

  1. Download your platform build and extract if zipped.
  2. Launch the starter script/binary: start_client.sh (Linux) or start_client.bat (Windows).
  3. Move with WASD/Arrows, buy/select weapons with keys 1/2/3/4.
  4. Press R to restart after death.
  5. Send feedback with timestamp + what happened + expected behavior.

Gameplay Notes

  • Goal: survive as long as possible while expanding and managing credits.
  • Combat modes in this build are preconfigured by launcher settings.
  • Spread and stun are useful for crowd control at mid-range.
  • Shift + direction lets you expand/select active cell.

Feedback Format

  • Category: crash, gameplay, UI, networking, performance
  • Repro: steps from launch to issue
  • Logs: include terminal output if possible
  • System: OS + GPU + approx FPS
